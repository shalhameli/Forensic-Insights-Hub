Forensic Insights Hub is a Python-based digital forensics toolkit designed to support **Windows Registry analysis** through an intuitive graphical user interface (GUI). The tool assists forensic investigators in collecting, analyzing, and reporting key system artifacts such as installed software, hardware details, user accounts, registry modification timelines, and registry exports.

The project focuses on simplifying complex forensic tasks by automating registry analysis using Python, making it accessible to both experienced investigators and beginners while maintaining forensic integrity.

---

## Project Objectives
- Automate Windows Registry analysis for forensic investigations
- Extract critical artifacts such as software, hardware, and user data
- Support timeline reconstruction using registry modification times
- Enable forensic-ready reporting with logging and hashing
- Provide a user-friendly GUI using Tkinter

---

## Technologies Used
- Python
- Tkinter (GUI)
- Winreg (Windows Registry access)
- SHA-256 hashing
- FPDF (report generation)
- Windows 10 / 11

---

## Tool Features
- Software enumeration (installed applications)
- Hardware component identification (CPU / GPU)
- User account and login information retrieval
- Registry key modification time analysis
- Registry viewing and navigation
- Registry export for forensic reporting
- Automated logging and hash generation for integrity
